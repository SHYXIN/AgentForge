# AgentForge

AI Agent 开发展示项目 - 多 Agent 协作 + RAG + 代码执行

## Agent skills

### Issue tracker

Issues and PRDs for this repo live as GitHub issues. Use the `gh` CLI for all operations. See `docs/agents/issue-tracker.md`.

### Triage labels

The skills use five canonical triage roles with default label names. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo with one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
